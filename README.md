🎟️ Event Management System
📌 Project Overview

This is a full-stack Event Management System that allows users to browse, explore, and manage different types of events such as technical events, music concerts, comedy shows, and more.
The system provides an interactive interface for users and a structured backend for handling event data efficiently.

🚀 Features / Functionality

View all available events
Categorization of events (Tech, Music, Comedy, Cultural, etc.)
Event details (name, location, date, description)
Add new events (admin functionality)
Responsive UI for better user experience
Backend API integration for dynamic data handling

🏗️ Project Architecture

The project follows a 3-tier architecture:

Presentation Layer (Frontend)
Built using React
Handles UI and user interaction
Application Layer (Backend)
Spring Boot REST APIs
Business logic and data processing
Data Layer (Database)
Stores event details
Managed using MySQL
